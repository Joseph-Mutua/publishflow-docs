# WordPress.org Assets

This directory stores starter assets for the WordPress.org plugin page.

If you publish through SVN, these files belong in the repository root `assets/` directory, not inside `trunk/`.

Included files:

- `banner-772x250.png`
- `banner-1544x500.png`
- `icon-128x128.png`
- `icon-256x256.png`
- `icon.svg`
- `screenshot-1.png`
- `screenshot-2.png`
- `screenshot-3.png`
- `screenshot-4.png`

The screenshot files should ideally be replaced with captures from a real local WordPress instance before the first public submission.
